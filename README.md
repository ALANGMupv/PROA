# PROA – Plataforma educativa

Este repositorio contiene la documentación y prototipo web del proyecto **PROA**, una plataforma educativa diseñada para facilitar la gestión académica entre alumnos, profesores y personal administrativo.

El proyecto incluye el diseño de la arquitectura de la información, historias de usuario, diagramas de flujo y wireframes para el **Producto Mínimo Viable (PMV)** de la plataforma.

---

## Descripción del proyecto

PROA es una plataforma que permite gestionar asignaturas, exámenes y calificaciones dentro de un entorno educativo.

Entre sus funcionalidades principales se encuentran:

* Inicio de sesión para diferentes tipos de usuario
* Visualización de asignaturas
* Realización y envío de exámenes
* Corrección automática de exámenes
* Visualización de calificaciones
* Gestión de alumnos y profesores por parte del personal administrativo

El sistema está diseñado para permitir que los estudiantes **realicen exámenes en la plataforma y reciban su calificación automáticamente**. 

---

## Funcionalidades principales

La plataforma contempla distintos roles de usuario:

### Alumno

* Visualizar asignaturas
* Realizar exámenes
* Consultar calificaciones
* Recibir notificaciones

### Profesor

* Crear exámenes
* Publicar evaluaciones
* Revisar resultados de los alumnos

### Personal administrativo (PAS)

* Gestionar asignaturas
* Matricular alumnos
* Vincular profesores a asignaturas

## Objetivo del proyecto

El objetivo de PROA es ofrecer una **plataforma educativa centralizada** que permita gestionar de forma eficiente las actividades académicas, facilitando la interacción entre estudiantes, profesores y personal administrativo.
